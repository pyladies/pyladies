Introduction

Bonjour !

Nous sommes un groupe qui vise à soutenir les femmes dans la communauté python (si vous vous identifiez en tant que femmes, nous aussi). Nous nous retrouvons régulièrement pour des "coding cafe" (tout niveau !) ainsi que des "talking bar".

Où nous trouver ?

  • Sur Twitter : @PyLadiesParis.
  • Sur Meetup : PyLadies-Paris.

Code de Conduite

En plus du code de conduite définit par Pyladies, nous suivons un code de confuite renforcé construit à partir de celui du Geek Feminism Wiki. Vous le trouverez dans la partie forum de notre page meetup.

Hello!
