Pyladies Budapest

We are a group for Python ladies of all levels of programming experience, in the NYC metro area.

 

Pyladies is an international mentorship group with a focus on helping more women become active participants and leaders in the Python open-source community. Our mission is to promote, educate and advance a diverse Python community through outreach, education, conferences, events and social gatherings.

 

PyLadies also aims to provide a friendly support network for women and a bridge to the larger Python world. Anyone with an interest in Python is encouraged to participate!

Különböző szinteken Python programozó lányok vagyunk Budapesten.

 

Pyladies egy nemzetközi mentor szervezet, amelynek célja, hogy több nő legyen a Python programozók közösségében. A küldetésünk, hogy hirdessük, edukáljuk és eseményeket szervezzük nekik.

 

PyLadiesnek szintén fontos célja, hogy egy barátságos és támogató környeztet alakítson ki a lányoknak, és hídat teremtsen a nemzetközi Python szervezetek között. Bárki aki érdeklődik a Python programnyelv iránt, bátran vegyen részt a programjaikon!
